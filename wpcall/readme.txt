wpcall helps users to make calls over WebRTC.

== Description ==

WebRTC calling plugin for WordPress. It uses SIP over WebSockets.


Features:

*   Uses SIP over WebSockets for signalling.
*   WebRTC is used for peer to peer communication.

== Installation ==

It is so similar to regular WordPress plugin installation,


1. Upload the plugin files to the `/wp-content/plugins/wpcall` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Plugin Name screen to configure the plugin


== Frequently Asked Questions ==

= Is it a yet another WebRTC plugin? =

WebRTC lacks of signalling solution. WPCall uses SIP protocol on top of that. So you can use a superset of SIP and WebRTC protocols. WPCall can be used to call any user supports SIP.
